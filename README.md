# Foobar

Pretty simple recipe app written in react, jsavascript.

### Built With
*React
*Javascript
*Framer motion
*Styled components
*Vite


## Installation

Use the package manager [npm](https://https://www.npmjs.com) to install the app.

```bash
##to install
npm i
```

## Usage

```
##to run locally
npm run dev

##to build for production
npm run 

```
